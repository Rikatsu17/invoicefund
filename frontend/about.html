About the project

We created blockchain Dapp project with idea that might be really helpful in real world examples.
If you are a small business owner, you might have faced the problem of working capital financing.
So, our project solves this problem by allowing businesses to create campaigns, where contributors can fund them with test ETH and receive reward tokens.
And we think that this project can be really helpful for small businesses
